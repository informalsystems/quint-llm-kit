# Quint Examples

This directory contains 75 Quint example specifications organized by category.

## Categories

- **classic** (20 examples) - Classic distributed algorithms
  - Bakery algorithm, Clock sync, Paxos, Raft, etc.
- **language-features** (15 examples) - Quint language feature demonstrations
- **cosmos** (14 examples) - Cosmos SDK and IBC specifications
- **tutorials** (6 examples) - Tutorial examples
- **solidity** (5 examples) - Solidity contract verification
- **games** (4 examples) - Game specifications
- **spells** (4 examples) - Reusable utility modules
- **cryptography** (3 examples) - Cryptographic primitives
- **puzzles** (2 examples) - Logic puzzles
- **cosmwasm** (1 example) - CosmWasm smart contracts
- **verification** (1 example) - Verification examples

## Index

The examples are indexed in `data/examples-index.json` with:
- Module names
- Type definitions
- Function signatures
- Action names
- Import dependencies
- Extracted descriptions
- Keywords for search
- Test presence

Run `npm run setup` to regenerate the index.
